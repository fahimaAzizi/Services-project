Principles of Design

About Principles of Design

The principles of design are the rules that guide designers in arranging elements in a composition. Key principles include:

  • Balance: Creating stability in design.
  • Contrast: Highlighting differences to draw attention.
  • Alignment: Arranging elements to create a visual connection.
  • Repetition: Reinforcing design consistency.
  • Proximity: Grouping related elements together.
  • Space: Using whitespace to improve readability.

About Adobe Photoshop

Adobe Photoshop is a raster graphics editor used for photo editing, digital art, and more. Key features include:

  • Layers for non-destructive editing.
  • Selection tools for precise edits.
  • Brush tools for painting and retouching.
  • Filters for creative effects.
