Introduction to C++

C++ is a beast-level programming language that powers high-performance systems, game engines, and real-time applications. Master it, and you master the code world!

What is C++?

C++ is a powerful, general-purpose programming language developed as an extension of the C programming language. It offers object-oriented features, low-level memory manipulation, and high performance.

Key Features:

  • Object-Oriented: Supports classes and objects.
  • Performance: Provides high-speed execution.
  • Flexibility: Used for system programming, game development, and more.
  • Standard Library: Includes data structures, algorithms, and input/output facilities.

Explanation:

  • #include <iostream>: Includes the Input/Output stream library.
  • using namespace std; Allows the use of standard library symbols.
  • int main(): Entry point of the program.
  • cout: Prints output to the console.
  • endl: Ends the line and flushes the output buffer.

#include <iostream>

    }

Variables & Data Types

Variables are containers for storing data values. Data types specify the type of data that a variable can hold.

Basic Data Types:

Each of these data types serves a unique purpose in C++ programming:

  • int: Used for whole numbers. Example: int count = 10;
  • float: Used for decimal numbers with single precision. Example: float temperature = 36.6;
  • double: Used for decimal numbers with double precision. Example: double largeValue = 12345.6789;
  • char: Used to store a single character. Example: char initial = 'B';
  • string: Used to store text. Requires #include <string>. Example: string message = "Hello";
  • bool: Used for true/false values. Example: bool isReady = true;

Control Structures

Master if, else, switch, for, while, and do-while loops.

If-Else Statement

Used for decision-making based on a condition.

        
        int x = 10;
        if (x > 5) {
            std::cout << "x is greater than 5";
        } else {
            std::cout << "x is less than or equal to 5";
        }

Switch Statement

Used when you need to select one option from many based on an integer or character value.

        
        int day = 3;
        switch (day) {
            case 1:
                std::cout << "Monday";
                break;
            case 2:
                std::cout << "Tuesday";
                break;
            default:
                std::cout << "Other day";
        }

For Loop

Used to execute a block of code multiple times with a counter.

        
        for (int i = 0; i < 5; i++) {
            std::cout << "Iteration: " << i << "\n";
        }

While Loop

Executes a block of code as long as a condition is true.

        
        int i = 0;
        while (i < 5) {
            std::cout << "Value: " << i << "\n";
            i++;
        }

Do-While Loop

Similar to `while`, but guarantees at least one execution.

        
        int i = 0;
        do {
            std::cout << "Value: " << i << "\n";
            i++;
        } while (i < 5);

Functions

Understand how to create and use functions for modular code.

What Are Functions?

Functions are blocks of code designed to perform specific tasks. They help organize your code into reusable and modular parts.

Types of Functions:

  • Built-in Functions: Predefined functions like cout, cin, etc.
  • User-defined Functions: Functions created by the programmer.

Syntax:

        
        returnType functionName(parameters) {
            // body of the function
        }

Example:

        
        #include <iostream>
        using namespace std;
        
        // Function to add two numbers
        int add(int a, int b) {
            return a + b;
        }
        
        int main() {
            int result = add(10, 20);
            cout << "Sum: " << result << endl;
            return 0;
        }

Explanation:

  • Return Type: Specifies the data type of the value the function will return (e.g., int).
  • Function Name: Identifies the function (e.g., add).
  • Parameters: Inputs passed into the function (e.g., a and b).
  • Function Body: Contains the statements to execute.
  • Return Statement: Sends the result back to the caller.

Pointers

Understand memory management with pointers.

Explanation:

A pointer is a variable that stores the memory address of another variable. Pointers allow direct memory manipulation, which is useful in systems programming, dynamic memory allocation, and data structures.

  • Declaration: int* ptr; declares a pointer to an integer.
  • Initialization: ptr = &x; assigns the address of x to ptr.
  • Dereferencing: *ptr accesses the value stored at the pointer's address.

Example:


    #include 
    using namespace std;
    
    int main() {
        int x = 10;
        int* ptr = &x;
        cout << "Value of x: " << x << endl;
        cout << "Pointer address: " << ptr << endl;
        cout << "Value at pointer: " << *ptr << endl;
        return 0;
    }

Object-Oriented Programming

Classes, objects, inheritance, and polymorphism.

Explanation:

Object-Oriented Programming (OOP) is a programming paradigm based on the concept of "objects", which can contain data and methods. Key concepts include:

  • Class: A blueprint for creating objects (e.g., class Car).
  • Object: An instance of a class (e.g., Car myCar;).
  • Inheritance: Allows a class to inherit properties and methods from another class.
  • Polymorphism: Allows methods to do different things based on the object it is acting upon.

Example:


    #include 
    using namespace std;
    
    class Car {
    public:
        string brand;
        int year;
    
        void display() {
            cout << "Brand: " << brand << ", Year: " << year << endl;
        }
    };
    
    int main() {
        Car myCar;
        myCar.brand = "Toyota";
        myCar.year = 2020;
        myCar.display();
        return 0;
    }

Templates in C++ allow you to write generic and reusable code. They enable functions and classes to operate with any data type without rewriting code for each type.

  • Class Templates: Create classes that work with any data type.

Example:

File Handling

Learn how to read and write files in C++.

Explanation:

File handling in C++ allows you to create, read, write, and close files using the fstream library.

  • ofstream: Used for writing to files.
  • ifstream: Used for reading from files.
  • fstream: Used for both reading and writing.

Example:


        #include 
        #include 
        using namespace std;

            ofstream myfile("example.txt");
            myfile << "Hello, File!";
            myfile.close();
        
            ifstream readfile("example.txt");

                cout << line << endl;
            }
            readfile.close();
        
            return 0;
        }

Memory Management

Dynamic memory allocation with new and delete.

Explanation:

In C++, dynamic memory allocation allows you to allocate memory at runtime using the new and delete operators.

  • new: Allocates memory on the heap and returns a pointer to it.
  • delete: Frees the memory allocated by new.

Example:


#include 
using namespace std;

int main() {
    int* ptr = new int; // dynamically allocate memory for an int
    *ptr = 42; // assign a value
    cout << "Value: " << *ptr << endl;
    delete ptr; // free allocated memory
    return 0;
}

Namespaces

Avoid name conflicts with namespaces.

Explanation:

Namespaces in C++ are used to organize code into logical groups and prevent name conflicts. They allow you to create distinct variable, function, and class names.

Example:


#include 
using namespace std;

namespace First {
    int value = 100;
}

namespace Second {
    int value = 200;
}

int main() {
    cout << First::value << endl;  // Access value from First namespace
    cout << Second::value << endl; // Access value from Second namespace
    return 0;
}

Preprocessor Directives

Macros, #define, #ifdef, and more.

Explanation:

Preprocessor directives in C++ are instructions processed before the actual compilation of code. They typically begin with a # symbol and are used for including files, defining constants, and conditional compilation.

Multithreading

Parallel programming with threads.

Explanation:

Multithreading in C++ allows the execution of multiple threads concurrently, enabling parallel programming. It helps in optimizing the use of CPU by running tasks simultaneously.

  • Thread Creation: Threads can be created using the std::thread class from the <thread> library.
  • Synchronization: Ensures that threads do not interfere with each other when accessing shared resources.

Example:


#include 
#include 
using namespace std;

void printMessage() {
    cout << "Thread is running" << endl;
}

int main() {
    thread t(printMessage); // Create a thread
    t.join(); // Wait for the thread to finish
    return 0;
}

// Simple To-Do List Application
#include 
#include 
using namespace std;

int main() {
    vector tasks;
    string task;
    char choice;

    do {
        cout << "Enter a task: ";
        cin.ignore();
        getline(cin, task);

        cout << "Add another task? (y/n): ";

    } while (choice == 'y');

}
