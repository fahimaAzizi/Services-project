Web Development Basics

Introduction

Welcome to the Web Development Basics course! This course is designed to teach you the fundamentals of HTML, CSS, and JavaScript. By the end, you'll be able to create a basic webpage and understand how these technologies work together.

Prerequisites

  • Basic computer knowledge.
  • A code editor (e.g., Visual Studio Code).
  • A web browser (e.g., Google Chrome).

Curriculum

HTML Basics

HTML (HyperText Markup Language) is the foundation of web development. Below is an introductory example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Welcome to My Page</h1>
  </body>
</html>

HTML Core Concepts

1. Text Formatting

<p>This is a paragraph.</p>
<strong>Bold Text</strong>
<em>Italic Text</em>
<u>Underlined Text</u>

2. Links

<a href="https://example.com">Visit Example</a>

3. Images

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

4. Lists

<ul>
  <li>Unordered Item 1</li>
  <li>Unordered Item 2</li>
</ul>
<ol>
  <li>Ordered Item 1</li>
  <li>Ordered Item 2</li>
</ol>

5. Tables

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>30</td>
  </tr>
</table>

6. Forms

<form>
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

7. Semantic Elements

<header>Header Section</header>
<article>Main Content</article>
<footer>Footer Section</footer>

8. Media Embedding

<video controls>
  <source src="video.mp4" type="video/mp4">
</video>
<audio controls>
  <source src="audio.mp3" type="audio/mp3">
</audio>

9. Inline vs Block Elements

Inline elements like <span> flow within text, while block elements like <div> create separate blocks.

10. Metadata

<meta charset="UTF-8">
<meta name="description" content="Learn HTML">

Explore these concepts to deepen your understanding of HTML. Experiment with different tags to see how they behave in a browser!

JavaScript Comprehensive Guide

JavaScript makes your web pages interactive and dynamic. Below is an introductory example:

console.log('Hello, World!');

JavaScript Core Concepts

1. Variables

let name = "John"; // Block-scoped variable
const age = 25; // Immutable variable
var city = "New York"; // Function-scoped variable

2. Data Types

let str = "Hello"; // String
let num = 42; // Number
let isTrue = true; // Boolean
let arr = [1, 2, 3]; // Array
let obj = { key: "value" }; // Object

3. Functions

function greet() {
  console.log("Hello, World!");
}
greet();

4. Conditional Statements

if (age > 18) {
  console.log("You are an adult.");
} else {
  console.log("You are a minor.");
}

5. Loops

for (let i = 0; i < 5; i++) {
  console.log(i);
}
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

6. DOM Manipulation

document.getElementById("example").innerText = "Updated Text!";
document.querySelector("h1").style.color = "blue";

7. Event Listeners

document.querySelector("button").addEventListener("click", function() {
  alert("Button clicked!");
});

8. Arrays

let fruits = ["Apple", "Banana", "Cherry"];
fruits.push("Orange"); // Add to the end
fruits.pop(); // Remove from the end
console.log(fruits[0]); // Access first element

9. Objects

let car = {
  make: "Toyota",
  model: "Corolla",
  year: 2021
};
console.log(car.make);

10. Promises

fetch('https://api.example.com')
  .then(response => response.json())
  .then(data => console.log(data));
  .catch(error => console.error(error));

11. Async/Await

async function fetchData() {
  try {
    let response = await fetch('https://api.example.com');
    let data = await response.json();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}
fetchData();

12. Classes

class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
  speak() {
    console.log(`Hi, I'm ${this.name}`);
  }
}
const john = new Person("John", 30);
john.speak();

13. Error Handling

try {
  throw new Error("Something went wrong!");
} catch (error) {
  console.error(error.message);
}

14. Modules

// Export
export const add = (a, b) => a + b;
// Import
import { add } from './math.js';
console.log(add(2, 3));

15. Debugging

Use console.log(), browser developer tools, and breakpoints to debug your code efficiently.

16. Best Practices

  • Write readable and well-documented code.
  • Use const and let instead of var.
  • Modularize your code for better reusability.

Explore these concepts to master JavaScript and build powerful, interactive web applications!
