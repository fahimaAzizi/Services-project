Advanced Java

Streams API

Functional programming for processing data.

  • filter: Select elements based on conditions.
  • map: Transform elements.
  • reduce: Combine elements to a single result.

Lambda Expressions

Simplify code with functional programming concepts.

  • Syntax: (parameters) -> expression
  • Used for: Implementing interfaces with single methods (Functional Interfaces).

Design Patterns

Reusable solutions to common software design problems.

  • Singleton: Ensure only one instance of a class exists.
  • Factory: Create objects without specifying the exact class.
  • Observer: Define a dependency between objects to notify changes.

Networking

Communicate between devices using Java.

  • Socket: Establish a client-server connection.
  • URL: Access resources from the web.
  • HttpURLConnection: Communicate using HTTP protocol.

JDBC

Connect Java applications to databases.

  • DriverManager: Load and register drivers.
  • Connection: Establish database connections.
  • Statement: Execute SQL queries.
